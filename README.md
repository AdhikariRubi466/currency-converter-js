# Currency Converter (NPR ↔ USD)

A simple and interactive web-based currency converter built using **HTML, CSS, and JavaScript**.  
This project allows users to convert money between **Nepalese Rupees (NPR)** and **US Dollars (USD)**.

The interface includes smooth UI animations, background images, and responsive input controls.

---

## Features

- Convert **NPR → USD**
- Convert **USD → NPR**
- Clean and modern UI design
- Smooth hover and focus animations
- Responsive layout
- Background styling with CSS
- Error handling for invalid inputs

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How It Works

1. Enter the amount you want to convert.
2. Select the currency you want to convert **from**.
3. Select the currency you want to convert **to**.
4. Click the **Convert** button.
5. The converted value will appear in the result box.

Conversion rate used:

```
1 NPR ≈ 0.0076 USD
```

Clone the repository:

```

```
